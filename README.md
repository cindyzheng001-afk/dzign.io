# dzign.io

An AI-powered interior design tool built with React, Vite, and the Google Gemini API.

## Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Configure API Key:**
    *   Open the **`.env`** file in the root directory.
    *   Paste your Google Gemini API key next to `VITE_API_KEY=`.
    *   Example: `VITE_API_KEY=your_key_here`
    *   *(If you don't have a key, get one here: https://aistudio.google.com/app/apikey)*

3.  **Run the App:**
    ```bash
    npm run dev
    ```

4.  **Open in Browser:**
    *   Click the link shown in the terminal (usually `http://localhost:5173`).
