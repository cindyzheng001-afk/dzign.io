# dzign.io

An AI-powered interior design tool built with React, Vite, and the Google Gemini API.

## Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run the App:**
    ```bash
    npm run dev
    ```

3.  **Open in Browser:**
    *   Click the link shown in the terminal (usually `http://localhost:5173`).
